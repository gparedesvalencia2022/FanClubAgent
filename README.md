# ⚽ Fan Club Québec City – AI Social Media Agent

## English

### 📌 Project Overview
This project is an AI-powered social media agent built with **C# / .NET**.  
It automatically generates and publishes football-related posts for a Facebook fan page.

The goal is to demonstrate:
- Automation skills
- Integration with external APIs
- Use of AI concepts (simulated or real)
- Clean architecture in .NET

---

### 🚀 Features

- ⚽ Generates posts based on football matches
- 🤖 AI-like content generation (real or simulated)
- 📘 Automatic publishing to Facebook (Graph API)
- 🔄 Loop through daily matches
- ⏱️ Anti-spam delay handling
- 🔐 Secure configuration using environment variables
